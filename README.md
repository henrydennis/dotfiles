# Dotfiles

Managed with [chezmoi](https://www.chezmoi.io/).

## Quick Start

To install these dotfiles on a new machine, run:

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply henrydennis
```

## What's Included

*   **Shell**: zsh
*   **Window Manager**: Aerospace
*   **Editors**: Zed
*   **Tools**: git, brew, karabiner

## Management

*   **Edit**: `chezmoi edit <file>`
*   **Apply**: `chezmoi apply`
*   **Update**: `chezmoi update`
